🚀 SETUP STEPS

1
Install Maki npm install @tialops/maki
2
Run dev server maki dev
3
Build 3 tilemaps maki tilemap → office_map, hackathon_map, escape_map
4
Enable debug to check collisions Set debug: true in maki.config.js
5
Record gameplay video & submit! GitHub repo + Devpost + Google Form
